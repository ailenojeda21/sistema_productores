## 1. OBJETIVO

Orientar al productor agropecuario sobre el propósito del sistema RUPAL, los módulos disponibles y la forma correcta de navegar por la aplicación.

## 2. ALCANCE

Aplica a todos los productores agropecuarios del departamento de Lavalle, Mendoza, que utilicen el sistema RUPAL para registrar y gestionar su información productiva.

## 3. ¿QUÉ ES RUPAL?

RUPAL (Registro Único de Productores Agropecuarios de Lavalle) es un sistema web desarrollado por la Municipalidad de Lavalle que permite a los productores agropecuarios:

- Registrar sus datos personales y de contacto.
- Gestionar sus propiedades rurales con geolocalización.
- Registrar cultivos por propiedad con variedades específicas.
- Declarar maquinaria e implementos agrícolas disponibles.
- Gestionar canales de comercialización y mercados de venta.
- Mantener un perfil completo que refleje su actividad productiva.

## 4. ESTRUCTURA DEL SISTEMA

El sistema se organiza en los siguientes módulos, accesibles desde el menú lateral izquierdo:

```
RUPAL
├── Inicio (Dashboard)
│   └── Resumen de completitud del perfil
├── Perfil
│   ├── Datos personales
│   ├── Avatar
│   └── Exportación de datos
├── Propiedades
│   ├── Listado de propiedades
│   ├── Crear / Editar propiedad
│   ├── Detalle de propiedad (con mapa)
│   └── Descarga de RUT
├── Cultivos
│   ├── Listado de cultivos
│   ├── Crear / Editar cultivo
│   └── Validación de hectáreas disponibles
├── Maquinarias
│   ├── Listado de maquinaria
│   ├── Crear / Editar maquinaria
│   └── Registro de implementos
└── Comercialización
    ├── Listado de comercios
    ├── Crear / Editar comercio
    └── Mercados y cooperativas
```

## 5. BARRA DE NAVEGACIÓN

[CAPTURA DE PANTALLA: Barra de navegación lateral izquierda con todos los módulos]

La barra lateral izquierda (color azul marino) muestra los seis módulos principales. Cada módulo puede mostrar un círculo de progreso que indica el porcentaje de completitud de esa sección.

| Elemento | Descripción |
|----------|-------------|
| Icono de menú (☰) | En móvil, abre/cierra el menú lateral |
| Módulo activo | Se resalta con fondo más claro |
| Círculo de progreso | Muestra el % de completitud del módulo |
| Cerrar sesión | En la parte inferior de la barra lateral |

## 6. BARRA SUPERIOR

[CAPTURA DE PANTALLA: Barra superior con nombre de usuario]

En la parte superior derecha se muestra:
- El nombre del productor autenticado.
- La inicial del nombre dentro de un círculo.
- Menú desplegable con opciones de perfil y cierre de sesión.

## 7. VISTA MÓVIL

[CAPTURA DE PANTALLA: Vista móvil con menú hamburguesa abierto]

En dispositivos móviles:
- La barra lateral se oculta inicialmente.
- Un botón de menú (☰) en la esquina superior izquierda abre un panel deslizante.
- Las tablas se reemplazan por tarjetas verticales.
- Los formularios se adaptan al ancho de la pantalla.

## 8. REQUISITOS PREVIOS

Antes de comenzar a usar RUPAL, el productor debe:

1. Tener conexión a Internet.
2. Contar con un navegador web actualizado (Chrome, Firefox, Edge o Safari).
3. Poseer una dirección de correo electrónico válida.
4. Tener a mano su DNI y datos de contacto.
5. Disponer de la documentación de sus propiedades (RUT, coordenadas, etc.).
